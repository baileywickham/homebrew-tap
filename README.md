# homebrew-tap

Homebrew tap for [baileywickham](https://github.com/baileywickham)'s apps.

## Usage

```bash
brew tap baileywickham/tap
brew install --cask artwall
```

Or in one step:

```bash
brew install --cask baileywickham/tap/artwall
```

## Casks

- **[artwall](Casks/artwall.rb)** — macOS menu bar app that sets art wallpapers from Artpaper packs. ([source](https://github.com/baileywickham/ArtWall))
